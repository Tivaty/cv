CSS syntax

selector{property:value;}

  • Properties of a selector should be arranged as alphabel, you can find all properties name on link
  • Use Class to apply different style to each tag which are same type:
    • In HTML add an atribute class="classname"
    • In CSS style add new Selector with name ".classname {property:value;}"
    • One class can apply to multiple tag
    • A tag can apply multiple classclass="classname1 classname2"
  • Use Ids to apply different style to each tag which are same type:
    • In HTML add an atribute id="IDname"
    • In CSS style add new Selector with name "#IDname {property:value;}"
    • An ID can be applied for only one tag
